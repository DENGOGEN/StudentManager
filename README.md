# Менеджер Личных Дел Студентов

Настольное приложение на платформе **WPF (C#)** для структурированного управления и организации личных дел студентов по группам.

## Особенности
* **Неблокирующий интерфейс:** Окно со списком студентов работает параллельно с основным деревом папок.
* **Автоматическое управление диском:** Структура каталогов синхронизируется с базой данных приложения.

## Инструкция по запуску
1. Откройте `StudentManager.sln` в Visual Studio 2022.
2. Нажмите `Ctrl + Shift + B` для сборки.
3. Нажмите `F5` для запуска приложения.
